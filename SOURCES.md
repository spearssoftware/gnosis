## Upstream Sources

### Theographic Bible Metadata
- Repository: https://github.com/robertrouse/theographic-bible-metadata
- License: CC-BY-SA 4.0
- Files used: people.json, places.json, events.json, verses.json, peopleGroups.json, easton.json
- Contains: biographical data, events, verse references, Easton's dictionary entries

### OpenBible Geocoding Data
- Repository: https://github.com/openbibleinfo/Bible-Geocoding-Data
- License: CC-BY 4.0
- Files used: ancient.jsonl, modern.jsonl
- Contains: geographic coordinates and place identification data

### scrollmapper/bible_databases
- Repository: https://github.com/scrollmapper/bible_databases
- License: MIT
- Files used: cross_references.txt (Treasury of Scripture Knowledge), kjv.db (KJV Bible text)
- Contains: ~340k verse-to-verse cross-references with vote scores, full KJV verse text

### Strong's Concordance
- Repository: https://github.com/openscriptures/strongs
- License: CC-BY-SA 3.0
- Files used: hebrew.json, greek.json
- Contains: Strong's Hebrew (8,674 entries) and Greek (5,524 entries) dictionary definitions, lemmas, transliterations

### Bible Dictionary Dataset
- Repository: https://github.com/neuu-org/bible-dictionary-dataset
- License: CC-BY 4.0
- Files used: smith/*.json, hastings/*.json, schaff/*.json, hitchcock/*.json
- Contains: Smith's, Hastings', Schaff's, and Hitchcock's Bible dictionaries (~20,900 entries)

### Bible Topics Dataset
- Repository: https://github.com/neuu-org/bible-topics-dataset
- License: CC-BY 4.0
- Files used: topics/**/*.json
- Contains: Nave's Topical Bible and Torrey's New Topical Textbook (~7,800 topics with verse references)

### Open Scriptures Hebrew Bible (morphhb)
- Repository: https://github.com/openscriptures/morphhb
- License: CC-BY 4.0
- Files used: 39 OSIS XML files (one per OT book)
- Contains: Hebrew Bible text with per-word morphology codes and Strong's numbers

### Open Scriptures Hebrew Lexicon
- Repository: https://github.com/openscriptures/HebrewLexicon
- License: CC-BY 4.0
- Files used: AugIndex.xml, LexicalIndex.xml, BrownDriverBriggs.xml
- Contains: Brown-Driver-Briggs Hebrew and English Lexicon with lexical index

### Macula Greek
- Repository: https://github.com/Clear-Bible/macula-greek
- License: CC-BY 4.0
- Files used: macula-greek-SBLGNT.tsv
- Contains: NT Greek text (SBLGNT) with per-word lemma, morphology, Strong's numbers, Louw-Nida domains

### Dodson Greek-English Lexicon
- Repository: https://github.com/biblicalhumanities/Dodson-Greek-Lexicon
- License: Public Domain
- Files used: dodson.csv
- Contains: 5,408 NT Greek lexicon entries keyed by Strong's numbers
